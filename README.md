# AI Code Assistant

A web-based AI programming assistant tool that helps developers integrate and interact with large language models for code generation, completion, and review.

## Features

- Web-based code editor interface
- Integration with AI language models
- Code generation and completion
- Project structure analysis
- One-click code integration
- Real-time AI assistance

## Tech Stack

- Backend: Python Flask
- Frontend: React
- AI Integration: OpenAI API (configurable)

## Setup

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

## Configuration

Create a `.env` file in the backend directory with your AI API credentials:
```
OPENAI_API_KEY=your_api_key_here
```
